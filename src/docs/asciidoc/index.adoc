== QR 코드 생성 API

QR 코드를 생성하여 URL과 함께 반환합니다.
컨퍼런스 ID와 세션 ID를 기반으로 QR을 생성하며, 세션 ID는 선택값입니다.

=== 요청 정보

- **HTTP Method:** `GET`
- **URL:** `/api/v1/admin/qr`
- **Query Parameters:**
- `conferenceId` (필수): 컨퍼런스 ID
- `sessionId` (선택): 세션 ID
- `url` (필수): QR에 포함될 URL

==== 요청 예시
include::{snippets}/qr-code-controller-test/qr-code_-create_-success/http-request.adoc[]

=== 응답 정보

- **Status:** `200 OK`
- **응답 내용:** QR 코드 정보 반환
